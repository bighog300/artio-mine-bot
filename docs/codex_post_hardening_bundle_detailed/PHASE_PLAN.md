
# Phase Plan

## Phase 1: Benchmark
- Build dataset of 50–100 sources
- Define metrics: recall, precision, extraction accuracy
- Build evaluation runner
- Output baseline report

## Phase 2: Extraction
- Page classification system
- Art-specific field extractors
- LLM fallback
- Validation + confidence scoring

## Phase 3: Crawling
- Smarter URL prioritization
- Incremental crawling
- JS-heavy site handling
- Domain throttling

## Phase 4: Operator Tools
- Review interface
- Correction workflows
- Diff + replay tools
- Source onboarding improvements

## Phase 5: Data Intelligence
- Entity normalization
- Cross-source linking
- Change detection
- Knowledge graph foundation
