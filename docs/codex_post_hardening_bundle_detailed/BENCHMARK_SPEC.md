
# Benchmark Specification

## Dataset
- 50–100 art-related websites
- Include galleries, museums, fairs, artist sites

## Annotations
- Page type
- Extracted fields
- Expected images
- Canonical URLs

## Metrics
- Page discovery recall
- Page classification accuracy
- Field extraction precision/recall
- Duplicate rate
- Crawl success rate

## Output
- JSON report
- Summary dashboard
- Error breakdown
