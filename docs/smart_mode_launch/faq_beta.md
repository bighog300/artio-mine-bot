# Smart Mode Beta FAQ

## Q1: What is the fastest way to get first results?
Start with the site homepage URL and let Smart Mode run without changing settings; first records should appear in under 5 minutes.

## Q2: What should I do if Smart Mode fails?
Use the retry action once. If it fails again, submit a bug with URL, timestamp, and screenshot.

## Q3: How do I know which records to trust?
Use confidence indicators and provenance details. Approve high-confidence records first.

## Q4: Can I use Smart Mode on any art-related site?
Yes, but quality varies by site structure and crawl accessibility.

## Q5: How do I report confusing UX?
Use the post-run survey and include exact step + expectation mismatch.

## Q6: Why can’t I view usage metrics?
Metrics are admin-only; request role upgrade from launch coordinator.

## Q7: What counts as launch success?
- 10/10 testers complete the script.
- CSAT > 4/5.
- Time-to-first-record < 5 minutes.
- Success rate > 85%.
