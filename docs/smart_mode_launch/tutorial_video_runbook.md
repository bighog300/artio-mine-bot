# Smart Mode 2-Minute Quickstart Video Runbook

## Goal
Produce a 2-minute screen recording with voiceover demonstrating: **paste URL -> run Smart Mode -> get records**.

## Runtime target
- 1:50 to 2:10 total duration.

## Scene-by-scene script

### 0:00 - 0:15 Intro
- Show app dashboard.
- Voiceover: "In this quickstart, we’ll launch Smart Mode and generate records from a single site URL."

### 0:15 - 0:45 Enter source
- Open Smart Mining page.
- Paste URL and name.
- Click start.
- Voiceover: "Paste your source URL, optionally name it, then start Smart Mode."

### 0:45 - 1:20 Watch progress
- Show status panel and progress updates.
- Voiceover: "Smart Mode crawls and classifies pages automatically. You’ll see live progress while records are created."

### 1:20 - 1:50 Review first records
- Open records table.
- Show confidence + provenance at a glance.
- Voiceover: "Within minutes, your first records are ready for review. Approve high-confidence items and edit edge cases."

### 1:50 - 2:00 Wrap
- Show completed run state.
- Voiceover: "That’s it—URL to structured records in under five minutes."

## Production checklist
- 1080p recording, 30fps minimum.
- Cursor highlight enabled.
- Captions burned in or included as sidecar.
- No sensitive tokens or private data visible.
- Include one intentional retry/error callout if available.
