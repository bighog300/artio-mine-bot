You are working in the Artio Mine Bot repository.

Implement a focused architecture evolution so the platform becomes source-agnostic by default and domain-specific only through the mapping workflow.

Do not stop at analysis. Make code changes, add/update tests, run them, and summarize exactly what changed.

## Product goal
The desired behavior is:
1. user enters a source URL,
2. system scans it generically for structure and content,
3. AI proposes structure, selectors, follow rules, and candidate record mappings,
4. admin approves/edits that mapping,
5. approved mapping becomes the runtime extraction contract,
6. mining produces records such as artists, venues, and events from the approved mapping.

The platform must not be configured specifically for art-related sources in core discovery/mining logic. Domain specificity should be chosen during mapping.

## Core principle
Make the system:
- generic in discovery
- configurable in mapping
- deterministic in mining
